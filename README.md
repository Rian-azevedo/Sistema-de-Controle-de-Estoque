# Sistema de Controle de Estoque (Python)

Projeto desenvolvido em Python com o objetivo de praticar lógica de programação, estruturas de dados e validações básicas, simulando um sistema simples de controle de estoque via terminal.

# Objetivo do Projeto

Treinar conceitos fundamentais de programação em Python, como:
    listas e dicionários
    estruturas de repetição
    condicionais
    funções
    validação de dados
    organização de código

O sistema simula operações comuns encontradas em sistemas reais de estoque.


#  Funcionalidades

    Cadastrar produtos (nome, preço e quantidade inicial)
    Listar produtos cadastrados
    Entrada de produtos no estoque
    Saída de produtos do estoque
    Alerta quando o produto possui poucas unidades
    Validações para evitar dados inválidos e estoque negativo


# Regras de Negocios

    Não é permitido cadastrar quantidades ou preços inválidos
    Não é possível adicionar ou remover quantidade menor ou igual a zero
    Não é permitido remover mais produtos do que o disponível em estoque
    O sistema alerta quando a quantidade de um produto fica abaixo de 20 unidades

# Tecnologias Utilizadas

    Python 3.13
